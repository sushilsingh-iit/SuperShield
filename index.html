Quick Installation

Get SuperShield running on your local machine in seconds. Requires Python 3.9+ and root/admin privileges for network interface access.

# 1. Clone the repository
git clone https://github.com/sushilsingh-iit/SuperShield.git
# 2. Navigate and install dependencies
cd SuperShield
pip install -r requirements.txt
# 3. Launch the firewall GUI
sudo python main.py gui

System Architecture

Six focused modules, each with a single responsibility, wired together via main.py to create a cohesive security layer.

01

Packet Sniffer

Scapy sniff() runs on a background thread, parsing incoming IP, TCP, and UDP headers seamlessly.

02

Firewall Rules Engine

Evaluates extracted headers against a JSON-loaded configuration and IP blocklist in real-time.

03

Action Handler

Determines whether to silently drop the packet or allow it to proceed to the OS networking stack.

04

Activity Logger

Writes structured allow/block decisions to firewall.log using Python's native logging framework.

05

Interface (GUI/CLI)

Pulls metrics from the Monitor module to update the Tkinter dashboard or terminal UI.
